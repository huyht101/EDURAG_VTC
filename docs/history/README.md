# Historical documents

Các file dưới thư mục này là bản thiết kế/gen thử cũ, được giữ để hiểu lịch sử và convention. Chúng đã bị database schema 1.0.0 và implementation hiện hành thay thế; không dùng làm specification triển khai.

Các điểm cũ có thể xuất hiện: schema account trước `auth_version`, seed password hard-code, tên field review cũ, database name cũ và tuyên bố tính năng chưa được code chứng minh.
